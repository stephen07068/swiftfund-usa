## 🎨 Customize Easily

### Change Colors
```typescript
// tailwind.config.ts
colors: {
  primary: { 600: '#your-color' }
}
```

### Change Text
```typescript
// components/sections/HeroSection.tsx
<h1>Your New Headline</h1>
```

### Change Email
```env
// .env
EMAIL_TO=your-email@example.com
```
